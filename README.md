# 🏡 House Price Prediction using Machine Learning

## 📌 Project Overview

This project implements a **Machine Learning regression model** to predict house prices based on several housing features such as income level, number of rooms, population, and geographical location.

The goal is to demonstrate how **machine learning algorithms can be used to analyze housing data and predict property prices**.

---

## 🎯 Objective

The objective of this project is to build a predictive model that estimates **house prices based on housing-related features** using regression techniques.

---

## 📂 Dataset

This project uses the **California Housing Dataset** provided by Scikit-learn.

Dataset characteristics:

* **20640 housing samples**
* **8 numerical features**
* Target variable: **House Price**

### Features in the Dataset

| Feature    | Description                |
| ---------- | -------------------------- |
| MedInc     | Median income in the area  |
| HouseAge   | Average house age          |
| AveRooms   | Average number of rooms    |
| AveBedrms  | Average number of bedrooms |
| Population | Population in the block    |
| AveOccup   | Average occupancy          |
| Latitude   | Latitude coordinate        |
| Longitude  | Longitude coordinate       |

Target Variable:

* **Price** → Median house value

---

## 🔎 Exploratory Data Analysis

The dataset was explored using several visualization techniques:

* Dataset summary statistics
* Feature distribution analysis
* Correlation heatmap
* Pairplot visualization

These steps help understand relationships between variables and house prices.

---

## 🤖 Machine Learning Model

The project uses the following algorithm:

**Linear Regression**

Linear regression models the relationship between independent variables (features) and the dependent variable (house price).

---

## 📊 Model Evaluation

The trained model was evaluated using:

* **Mean Squared Error (MSE)**
* **R² Score**

These metrics measure how accurately the model predicts house prices.
